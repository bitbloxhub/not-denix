# not-denix: It's not [`denix`](https://github.com/yunfachi/denix)!
Well, it's /like/ denix, but based on [`flake-parts`](https://flake.parts/)!

# Quickstart
Run src_bash{nix flake init -t github:bitbloxhub/not-denix} in a directory.  
This is the example config, which does not scale well but has a simple introduction to the concepts.
